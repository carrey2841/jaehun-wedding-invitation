# 모바일 청첩장

- **폰트**: 고운바탕
- **톤**: 그레이 + 브라운
- **참고 시안**: [아름(Areum) 아르테미스 청첩장](https://www.areum.co.kr/invitation/wedding-invitation-artemis) — 커버(이름·날짜) → 일시·장소·오시는 길 → 갤러리 → 연락처·부모님·계좌 순으로 구성

## 기술 스택

| 구분 | 선택 |
|------|------|
| **프레임워크** | React 19 |
| **언어** | TypeScript |
| **빌드** | Vite 7 |
| **상태 관리** | Zustand (경량, 타입 안전, 컴포넌트 단위 활용에 적합) |
| **스타일** | CSS (전역 + CSS Modules 또는 스코프된 스타일) |

### 디렉터리 구조 (컴포넌트 단위)

```
src/
├── components/     # 재사용 UI 컴포넌트
├── sections/       # 청첩장 섹션 단위 (Cover, Gallery, Contact 등)
├── stores/         # Zustand 스토어
├── types/          # 공용 타입
├── styles/         # 전역 스타일, 테마
└── assets/         # 이미지 등 정적 자산
```

## 갤러리 사진

`gallery/` 폴더에 아래 이름으로 40장 넣어 주세요.

- `01.jpg`, `02.jpg`, `03.jpg`, … `40.jpg`

없는 번호는 자동으로 placeholder가 표시됩니다.

## placeholder로 나중에 채울 항목

- **커버**: 신랑·신부 이름, 결혼일, 서브타이틀
- **일시·장소**: 날짜·시간, 장소명, 주소, 지도 링크, 예식 순서(선택)
- **연락처**
- **부모님 성함** (신랑/신부)
- **마음 전하실 곳** (계좌)

## 로컬 확인

```bash
cd wedding-invitation
# 간단한 서버 (Python 3)
python3 -m http.server 8080
# 브라우저에서 http://localhost:8080
```

## 배포

Cloudflare Pages에 `wedding-invitation` 폴더를 업로드하거나, Git 저장소 루트를 이 폴더로 설정하면 됩니다.

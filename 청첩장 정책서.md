# 청첩장 정책서

## 1. 커버

- **대표 이미지**: 커버 상단 대표 이미지 노출
- **예식 일시**: 디자인을 이쁘게 꾸며서 노출
- **위치 + 시간**: 장소와 시간 정보 표시

---

## 2. Invitation (모시는 말씀)

- **내용은 나중에 값을 바꿀 수 있게** 구현 (스토어/설정으로 수정 가능)

**기본 내용:**

```
모시는 말씀

사랑으로 키워온 자녀가 좋은 인연을 만나
새로운 가정을 이루게 되었습니다.
부디 오셔서 두 사람의 시작을 축복해 주시고,
앞으로의 길이 행복으로 가득할 수 있도록
따뜻한 응원과 격려를 보내주시면 감사하겠습니다.

이관섭•이미라 장남 이재훈
이승복•김현주 장녀 이영주
```

- **연락하기 버튼** 노출
- 버튼 클릭 시 **팝업** 표시
  - 혼주 연락처, 신랑·신부에게 **문자** / **전화** 버튼
  - 전화번호는 **화면에 노출하지 않고**, 버튼만 눌러서 전화/문자 앱으로 이동

---

## 3. 캘린더

- **문구 노출**: "2026년 05월 16일 토요일 오후 5시"
- **캘린더에 일정 추가** 기능 (일정 등록/캘린더 앱 연동)

---

## 4. 갤러리

- 노출할 이미지: **40~50장** 예정
- **3×4 배열**로 **01.png ~ 12.png**까지 첫 화면에 노출
- **더보기** 버튼을 아래에 배치
- 더보기 클릭 시 **12장씩 추가** 로드 (13~24, 25~36, …)
- **사진 클릭** → **큰 팝업(라이트박스)**으로 사진 표시
- **닫기 버튼** 클릭 시 팝업 닫기

---

## 5. 오시는 길

- **지도**: **네이버 지도** 이용하여 표시
- **예식장 위치**: 서울특별시 송파구 천호대로 996 라비니움 이야
- **앱 연동 버튼**: 네이버지도, 카카오맵, T맵 — 각 앱으로 열기
  - 버튼은 **각 앱 로고만** 노출
- **placeholder** (추후 기입): 지하철, 버스, 자가용, 주차 정보

---

## 6. 마음 전하실 곳

**문구:**

```
마음 전하실 곳
참석이 어려우신 분들은
축하의 마음을 전달해 주세요.
```

- **신랑측 계좌**: 신랑, 신랑 아버지, 신랑 어머니 — 각 계좌 따로 설정
- **신부측 계좌**: 신부, 신부 아버지, 신부 어머니 — 각 계좌 따로 설정
- **계좌번호 복사** 버튼을 계좌번호 옆에 작게 배치
- **신랑측 / 신부측** 영역은 **접기(fold)** 가능한 컴포넌트로 구현 (기본 접힌 상태)
